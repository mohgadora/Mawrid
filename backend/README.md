# الواجهة الخلفية (Backend API)

خدمة REST API واحدة بمعمارية Modular Monolith (FastAPI) بثماني وحدات معزولة الحدود: الهوية وKYC، الكتالوج والبحث، التسعير وRFQ، الطلبات والمرتجعات، الدفع والمحفظة، التوصيل والأسطول، محرك أسعار السوق، والمهام الخلفية (Celery).

- التصميم المعماري الكامل: `docs/sdlc/7.DES-01_HLD_AR.docx`
- مخطط قاعدة البيانات التنفيذي: `docs/sdlc/8.DES-02_schema.sql`
- عقد الواجهات (OpenAPI 3): `docs/sdlc/9.DES-03_openapi.yaml`

لم يبدأ التطوير بعد.
