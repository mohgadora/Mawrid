# بوابة المورد (Supplier Portal)

تطبيق المورد: إدارة المنتجات والمخزون والأسعار والشرائح، والرد على طلبات عروض الأسعار (RFQ)، ومتابعة التسويات.

- المرجع الوظيفي: `SRS-CAT`, `SRS-PRC`, `SRS-PAY-03` في `docs/sdlc/5.REQ-02_SRS_AR.docx`
- المرجع المعماري: قسم "مسؤوليات المكونات" في `docs/sdlc/7.DES-01_HLD_AR.docx`

ملاحظة: صفحة عرض عامة لملف المورد موجودة حالياً ضمن `apps/web` (`/supplier/[id]`)، وهي ليست بديلاً عن بوابة إدارة المورد الكاملة.

لم يبدأ التطوير بعد.
