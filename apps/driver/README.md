- المرجع الوظيفي: `SRS-DLV-01` إلى `SRS-DLV-04` في `docs/sdlc/5.REQ-02_SRS_AR.docx`
- المرجع المعماري: قسم "مسؤوليات المكونات" في `docs/sdlc/7.DES-01_HLD_AR.docx`

لم يبدأ التطوير بعد.
