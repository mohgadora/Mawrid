# Mawrid

منصة تجارة الجملة الإلكترونية B2B / B2C

## هيكلة المستودع

المشروع منظّم كـ monorepo، وكل تطبيق في مجلد منفصل تحت `apps/`، خلف واجهة خلفية واحدة في `backend/`:

| المجلد | التطبيق | الحالة |
| --- | --- | --- |
| `apps/web` | تطبيق المشتري (ويب) — تصفح الكتالوج، السلة، الطلبات، الحساب | قيد التطوير |
| `apps/admin` | لوحة الإدارة | لم يبدأ |
| `apps/driver` | تطبيق السائق (التوصيل) | لم يبدأ |
| `apps/supplier` | بوابة المورد | لم يبدأ |
| `backend` | واجهة REST API خلفية واحدة (Modular Monolith) | لم يبدأ |
| `docs/sdlc` | مستندات دورة حياة تطوير البرمجيات (الميثاق، BRD، SRS، HLD، مخطط قاعدة البيانات، عقد OpenAPI، خطط الاختبار...) | مكتملة |

راجع `docs/sdlc/7.DES-01_HLD_AR.docx` للتفاصيل المعمارية الكاملة، و`docs/sdlc/5.REQ-02_SRS_AR.docx` للمواصفات الوظيفية لكل تطبيق.

## التطوير

المشروع يستخدم pnpm workspaces. لتشغيل تطبيق الويب:

```
cd apps/web
pnpm install
pnpm dev
```
